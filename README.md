# 🌿 EcoCode: AI-Powered Sustainability Auditor

An automated tool that analyzes code (Python, Java, HTML, C) to improve energy efficiency and performance using Groq AI.

Features
* **Multi-Language Support**: Optimizes structure and logic across different file types.
* **Carbon Impact Report**: Calculates estimated CO2 savings per file load.
* **Safety First**: Includes a "Strategy B" execution test for Python scripts.
* **Complexity Analysis**: Dynamic Time/Space complexity reports.
